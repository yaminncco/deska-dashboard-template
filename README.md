## Quick Start

Clone this repository

```bash
git clone https://github.com/yaminncco/deska-dashboard-template.git
```

Install dependencies

```bash
pnpm install
```

Start development server

```bash
pnpm dev
```

Open http://localhost:3000 with your browser to see the result.
